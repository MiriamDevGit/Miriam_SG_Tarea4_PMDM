##Aplicación Spyro##
#Introducción#
Aplicación inspirada en el mundo de Spyro. Detalla información sobre los personajes y mundos de este juego.
#Características principales#
Puedes ver un listado de personajes, mundos y colecciones. Información sobre el desarrollador y contiene una guía para iniciarte en la app.
#Instrucciones de uso#
Clonar repositorio Si ya tienes un proyecto abierto, ve a Archivo > Nuevo > Proyecto desde el Control de Versiones > Git. En el campo URL, pega la URL del repositorio de GitHub. Clonar: Haz clic en "Clonar"
